# Kh Shafin Farhan - Portfolio Website

## Quick Start

**Simplest method:** Double-click `index.html` to open in your browser. Done!

## Running Locally (Optional)

For better performance, use a local server:

**Python:**
```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`

**Node.js:**
```bash
npx http-server
```

**VS Code:**
Install "Live Server" extension, right-click `index.html` â†’ "Open with Live Server"

## Features
- Responsive design for all devices
- Interactive 3D globe with satellite imagery showing countries visited
- Smooth scrolling navigation
- GitHub project links

## Deploy for Free

**GitHub Pages (Recommended):**
1. Create repository: `username.github.io`
2. Upload all files
3. Live at: `https://username.github.io`

**Alternatives:** Netlify, Vercel, Cloudflare Pages

## Files
- `index.html` - Main portfolio page
- `globe.html` - Interactive 3D Earth
- `styles.css` - Styling
- `script.js` - Navigation & interactions

## Requirements
- Modern browser with WebGL (for 3D globe)
- Internet connection (for satellite textures from CDN)
